Start Coding with GitHub

  1. Create a GitHub account at github.com.
  2. Click on "New Repository" and name it something like learn-code.
  3. Upload your index.html file.
  4. Go to Settings > Pages and enable GitHub Pages by selecting the main branch.
  5. Your site will go live at: https://yourusername.github.io/learn-code
